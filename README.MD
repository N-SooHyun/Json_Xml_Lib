# Json_Xml Library

이 프로젝트는 **JSON**과 **XML** 데이터를 다룰 수 있는 로우레벨 기반의 C/C++ 라이브러리입니다.

- `.lib` 및 `.dll` 파일을 직접 생성하여, 다양한 플랫폼에서 활용 가능하도록 설계되었습니다.
- 외부 종속성을 최소화하고, 시스템 단에서도 작동 가능하도록 최대한 로우레벨에서 개발을 진행하고 있습니다.
- 본 라이브러리는 학습과 멀티플랫폼 실험 목적을 겸하여 개발되었습니다.
